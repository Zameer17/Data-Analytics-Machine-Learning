## 🧠 What is This Repository?

> A hands-on collection of **ML tasks and mini-projects** covering the complete data science pipeline — from raw messy data all the way to trained models and meaningful visualizations. Built during my **Data Science Internship at Syntech Technologies** and continued through self-learning.

```python
# The end-to-end pipeline followed in every project
raw_data
  → 🧹 data_cleaning
  → 🔍 exploratory_data_analysis
  → ⚙️  feature_engineering
  → 🤖 model_training
  → 📐 evaluation
  → 📊 visualization
  → 💡 insights  ✅
```

## 🔬 Workflow Covered

<div align="center">

| 🔢 Stage | 📝 What It Covers |
|:---:|:---|
| 1️⃣ | **Data Collection** — Load datasets from CSV or public sources |
| 2️⃣ | **Data Cleaning** — Handle nulls, duplicates, outliers & inconsistencies |
| 3️⃣ | **EDA** — Distributions, correlations, patterns & anomaly detection |
| 4️⃣ | **Feature Engineering** — Encoding, scaling, new feature creation |
| 5️⃣ | **Model Training** — Regression, Classification & Clustering algorithms |
| 6️⃣ | **Model Evaluation** — Accuracy, R², RMSE, Confusion Matrix, ROC |
| 7️⃣ | **Hyperparameter Tuning** — GridSearchCV, cross-validation |
| 8️⃣ | **Visualization** — Charts, heatmaps & insight storytelling |

</div>
